# 🔥 LankaGas 🏠  
### Real-Time Gas Availability Tracker for Sri Lanka

**LankaGas** is a crowd-powered web application that helps users find Litro and LAUGFS gas availability across Sri Lanka in real-time. It uses live GPS verification and user updates to provide accurate queue and stock information.

---

## 🌐 Live Demo  
https://lankagas.netlify.app/

---

## 🚀 Features  

- 📍 Live Location Verification  
  Ensures users are physically at the gas shop before submitting updates.

- 🔄 Real-Time Crowd Updates  
  Users can submit queue status and availability instantly.

- 📊 Queue Status Indicator (0% - 100%)  
  Easily understand how crowded a location is.

- 🛢️ Filter by Cylinder Type  
  View availability based on different gas cylinder sizes.

- 🗺️ Google Maps Navigation  
  Get direct directions to the selected gas station.

- 🧹 Daily Data Reset  
  Keeps information fresh and relevant every day.

---

## 🛠️ Tech Stack  

- Frontend: HTML, CSS, JavaScript  
- Backend/Database: Firebase  
- Hosting: Netlify  
- APIs: Geolocation API, Google Maps API
